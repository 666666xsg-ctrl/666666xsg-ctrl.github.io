# 个人履历展示网站 - 设计规范

## 1. 概念与愿景

一个高端、简洁的个人履历展示网站，以现代简约主义为核心设计理念。通过大量留白、精致排版和微妙动效，营造专业、可信赖的个人品牌形象。网站如同精心设计的名片，让访客在第一时间感受到主人的专业品味与独特个性。

## 2. 设计语言

### 美学方向
- **风格**: 现代简约主义 + 高端商务感
- **灵感**: Apple 官网的产品展示、Dribbble 高质量作品集
- **特点**: 极简留白、精致细节、优雅动效、专业气质

### 配色方案
```
主色调 (Primary):     #1a1a2e    - 深邃墨蓝，用于标题和重点元素
次要色 (Secondary):   #4a4a68    - 柔和灰紫，用于次级文字
强调色 (Accent):      #e94560    - 活力玫红，用于 CTA 和亮点
背景色 (Background):  #fafafa    - 近纯白，页面主背景
卡片背景:             #ffffff    - 纯白卡片背景
边框色:              #eee       - 极淡灰，用于分隔线
文字色:               #333       - 深灰正文文字
```

### 字体系统
```
标题字体: "Noto Serif SC", Georgia, serif - 优雅衬线体，体现高端感
正文字体: "Inter", -apple-system, sans-serif - 现代无衬线体，高可读性
英文装饰: "Playfair Display", serif - 优雅英文装饰字体
字体大小层级:
  - 超大标题: 4rem (64px)
  - 大标题: 2.5rem (40px)
  - 中标题: 1.5rem (24px)
  - 正文: 1rem (16px)
  - 小文字: 0.875rem (14px)
```

### 空间系统
```
基础单位: 8px
间距层级:
  - xs: 8px   (元素内紧凑间距)
  - sm: 16px  (元素内标准间距)
  - md: 24px  (组件间间距)
  - lg: 48px  (区块间间距)
  - xl: 80px  (大区块间距)
  - 2xl: 120px (页面级间距)
最大内容宽度: 1200px
```

### 动效哲学
- **原则**: 克制而精致，为体验加分而不喧宾夺主
- **入场动画**: 元素从下方 fade-in，opacity 0→1, translateY 20px→0, 600ms ease-out
- **交错动画**: 多元素时，每项延迟 100ms 依次入场
- **悬停效果**: 轻微上浮 + 阴影加深，200ms ease
- **链接交互**: 下划线从左向右展开，300ms
- **页面过渡**: 整体淡入，400ms

### 视觉资产
- **图标**: Lucide React - 线性风格，stroke-width: 1.5
- **装饰**: 极简几何线条、微妙的渐变光晕
- **图片策略**: 优先使用高质量抽象图案，避免低质量素材

- **桌面端 (>1024px)**: 完整双栏布局

- **移动端 (<768px)**: 全宽单栏，增大触控区域

## 4. 功能与交互

### 核心功能

#### 4.1 导航系统
- 固定顶部导航栏，滚动时添加背景模糊效果
- 点击导航项平滑滚动到对应区块
- 移动端汉堡菜单，侧滑展开

#### 4.2 英雄区域
- 大号姓名标题，带打字机效果的职位描述
- CTA 按钮："了解更多" 滚动到关于区域
- 微妙的背景装饰动画

#### 4.3 关于我区域
- 左侧展示个人抽象头像或装饰图形
- 右侧展示详细自我介绍
- 入场时左右交错动画

#### 4.4 项目展示区域
- 网格卡片布局
- 每个项目卡片包含：
  - 项目封面图（抽象图案占位）
  - 项目名称
  - 项目简介
  - 技术标签
  - 查看详情链接
- 悬停时卡片微浮起，显示更多信息
- 项目内容预留接口，后续可配置

#### 4.5 联系方式区域
- 突出展示邮箱地址（可点击复制）
- 小红书账号链接（带图标）
- 其他社交媒体链接（GitHub、LinkedIn 等）
- 悬停时图标放大效果

#### 4.6 页脚
- 简洁版权信息
- 回到顶部按钮

### 交互细节
- **邮箱复制**: 点击邮箱区域显示"已复制"提示，1.5秒后消失
- **链接悬停**: 颜色过渡 + 下划线动画
- **卡片悬停**: translateY(-4px) + box-shadow 增强
- **滚动监听**: 各区块内容在进入视口时触发动画

## 5. 组件清单

### 5.1 Navigation 导航栏
- **状态**: 默认 | 滚动后（有背景）| 移动端展开
- **样式**: 固定顶部，高度 64px，logo 左侧，导航链接右侧

### 5.2 HeroSection 英雄区域
- **状态**: 加载中（标题打字动画）| 完成
- **内容**: 大标题 + 副标题 + CTA 按钮

### 5.3 AboutSection 关于区域
- **状态**: 默认 | 入场动画中 | 入场完成
- **布局**: 左右分栏（桌面）/ 上下堆叠（移动）

### 5.4 ProjectCard 项目卡片
- **状态**: 默认 | 悬停 | 空状态（显示添加提示）
- **内容**: 封面 + 标题 + 描述 + 标签

### 5.5 ProjectSection 项目区域
- **状态**: 默认 | 空状态（无项目）| 有项目
- **布局**: 2列网格（桌面）/ 1列（移动）

### 5.6 ContactSection 联系区域
- **状态**: 默认 | 邮箱已复制（显示提示）
- **内容**: 邮箱 + 小红书 + 其他社交图标

### 5.7 SocialIcon 社交图标
- **状态**: 默认 | 悬停（放大）| 点击
- **类型**: 小红书、邮箱、GitHub、LinkedIn 等

### 5.8 Footer 页脚
- **内容**: 版权文字 + 回到顶部按钮
- **样式**: 居中，简洁分隔线

## 6. 技术方案

### 框架选择
- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **样式方案**: Tailwind CSS
- **动画库**: 内联 CSS 动画 + Intersection Observer
- **图标**: Lucide React

### 项目结构
```
portfolio-website/
├── index.html
├── package.json
├── tsconfig.json
├── vite.config.ts
├── tailwind.config.js
├── postcss.config.js
└── src/
    ├── main.tsx
    ├── App.tsx
    ├── index.css
    ├── components/
    │   ├── Navigation.tsx
    │   ├── HeroSection.tsx
    │   ├── AboutSection.tsx
    │   ├── ProjectSection.tsx
    │   ├── ProjectCard.tsx
    │   ├── ContactSection.tsx
    │   └── Footer.tsx
    ├── hooks/
    │   └── useIntersectionObserver.ts
    ├── data/
    │   └── portfolio.ts          # 个人信息和项目数据
    └── types/
        └── index.ts              # TypeScript 类型定义
```

### 数据管理
- 所有个人信息集中在 `src/data/portfolio.ts` 配置
- 项目数据为数组结构，便于后续添加
- TypeScript 类型确保数据完整性

### 性能优化
- 图片懒加载
- 动画使用 transform 和 opacity（GPU 加速）
- 代码分割（如需要）
